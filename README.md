# MATLAB Linear Equation Solver
## Description
The MATLAB Linear Equation Solver is a simple application developed using MATLAB to solve systems of linear equations efficiently. It provides an easy-to-use interface where users can enter coefficients and constants to compute solutions accurately. This project demonstrates MATLAB programming fundamentals, matrix operations, and basic GUI development.
## Features
- Solve systems of linear equations
- User-friendly interface
- Fast and accurate calculations
- Input validation
- Displays computed solutions clearly
- Simple MATLAB implementation

- Matrix Operations

## How to Run
1. Open MATLAB.
2. Open the `LinearEquationSolver.mlapp` (or `.m`) file.
3. Click **Run**.
4. Enter the coefficients and constant values.
5. Click the **Solve** button.
6. View the calculated solution.
